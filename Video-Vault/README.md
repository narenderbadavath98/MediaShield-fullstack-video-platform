# MediaShield - Secure Video Processing & Streaming

**MediaShield** is a comprehensive full-stack application for secure video management, automated sensitivity analysis, and seamless streaming.

## Core Features
- **Secure Video Upload:** Support for MP4/MOV with size validation.
- **Sensitivity Processing:** Real-time AI analysis for safe/flagged content.
- **HTTP Range Streaming:** Secure playback using 206 Partial Content.
- **RBAC:** Multi-tenant architecture with Viewer, Editor, and Admin roles.
- **Real-Time Progress:** Live status updates via Socket.io.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express, MongoDB.
- **Infrastructure:** FFmpeg for processing, JWT for Auth.
